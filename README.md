# color-generator
... a simple color palette/hue generator! aka a quick mini project speedrun [![Athena Award Badge](https://img.shields.io/endpoint?url=https%3A%2F%2Faward.athena.hackclub.com%2Fapi%2Fbadge)](https://award.athena.hackclub.com?utm_source=readme)

check it out [here <3](https://pepper-0.github.io/color-generator/)

### *features*
- generate!: generate a (set of) random color(s)
- add another color: add up to 5 colorboxes that display on the site concurrently
    - you will be notified if you exceed 5 colorboxes.
- remove a color: remove most recently added colorbox
